# 黑洞智影插件使用说明

黑洞智影直播插件封装了原生的live-pusher组件，使用系统原生组件，可以为没有直播资质的业务方提供直播服务，利用黑洞智影直播插件提供的各项api，业务方可以方便、快速地实现进出直播间、IM消息收发和音视频播放功能，可以快速地完成直播间的搭建。
